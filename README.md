# 법인영업 인생역전 - The Real Class

법인영업 실습 교육 랜딩 페이지입니다.

## 특징

- 🎨 모던한 다크 테마 UI
- 📅 CSV 파일 기반 동적 일정 관리
- 📱 반응형 디자인
- ✨ 부드러운 스크롤 애니메이션

## 파일 구조

- `index.html` - 메인 랜딩 페이지
- `lecture_schedule.csv` - 일정 데이터 (CSV 형식)

## 일정 업데이트

`lecture_schedule.csv` 파일을 수정하면 웹페이지의 일정이 자동으로 업데이트됩니다.

CSV 형식:
```
장소,시작일,종료일,잔여
서울,2025-05-22,2025-05-24,6
광주,2025-05-29,2025-05-31,3
부산,2025-07-10,2025-07-12,10
```
